# Setup (Windows)

This app is Rust + `eframe/egui` (not Tauri).

Use `README.md` as the primary guide for:
- prerequisites
- local run (`cargo check`, `cargo run`)
- release EXE build
- installer build
- GitHub Releases flow

Quick commands:

```powershell
cargo check
cargo run
cargo build --release
.\scripts\build-installer.ps1 -BuildName local-test1
```
